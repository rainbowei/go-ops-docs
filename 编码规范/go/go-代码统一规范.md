## 2. Code Formatter

### GoLand
统一使用 `gofmt` 自带的插件进行代码格式化

setting->Tools->File Watchers 选择 gofmt

### VS Code

请使用 [go-vscode.gofmt.settings.json](编码规范/go/go-vscode.gofmt.settings.json) 文件
